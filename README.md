# MATH 3180 — Mathematics for Machine Learning

This repository contains materials, notes, and implementations developed throughout **MATH 3180: Mathematics for Machine Learning**.

## Overview

This course focuses on the mathematical foundations that underpin modern machine learning methods. Topics emphasize the theory and intuition behind algorithms rather than just implementation.

## Topics Covered

* Linear Algebra (vectors, matrices, eigenvalues, eigenvectors)
* Probability and Statistics
* Optimization techniques
* Dimensionality reduction (e.g., PCA)
* Mathematical modeling for machine learning

## Purpose

The goal of this repository is to:

* Reinforce theoretical understanding through practice
* Provide reference implementations and examples
* Serve as a foundation for more advanced ML coursework and research

## Applications

Concepts from this course are directly applicable to:

* Machine Learning and AI
* Data Science
* Computer Vision
* Research in applied mathematics and engineering
